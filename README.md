## <img width="40" height="40" alt="tools_7628737" src="https://github.com/user-attachments/assets/6950d8b3-5eff-498a-aa09-15df82d78123" /> Как настроить

У Вас должен быть установлен [Python](https://www.python.org/downloads/).

Также в командной строке нужно будет установить набор зависимойстей командой:
```
pip install -r requirements.txt
```
Скрипт будет публиковать посты в группу телеграма через бота. Взаимодействовать с ботом он будет через ключ(токен). Если у Вас ещё нет группы в телеграме, тогда создайте и настройте её. Небольшой [гайд](https://way23.ru/%D1%80%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F-%D0%B1%D0%BE%D1%82%D0%B0-%D0%B2-telegram.html) каким образом создать бота и получить ключ.

После того как создана группа, бот добавлен в группу и получен ключ от него, создайте файл `.env` и запишите в него данные в следующем формате:
```
BOT_TOKEN='Ключ от Вашего бота'
CHAT_ID='@ID Вашей группы'
```
> Значения указываются обязательно в кавычках.

> ID группы можно узнать открыв информацию о группе, перед ID поставьте @.
